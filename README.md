# sharE

## サイト概要
*ライブやフェスなどの情報共有サイト

### サイトテーマ
*ライブやフェスなど情報を共有し、コミュニティを広げてもらう

### テーマを選んだ理由
前職でライブやフェスに関わる仕事をしていたため、その知識や関心を生かせるものを作りたかったため

### ターゲットユーザ
ライブやフェスに行く人々

### 主な利用シーン
ライブの期待度や評価、情報、感想の共有
